Git Branching

  • git branch: shows all the branches that currently exist in a repository (default is set to master).
  • git branch {branch_name}: Allows you to create a branch.
  • git checkout: change locally from working on one branch to another.
  • git push --set-upstream origin {branch_name}: allows you to push branch to github.
  • git merge: merge a branch into master branch.
  • Remote: online version of the repository
  • git pull
  • Forks: make changes to a version of a repository
